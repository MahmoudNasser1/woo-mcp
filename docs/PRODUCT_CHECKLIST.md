# ✅ Fix Zone Product Checklist

## 📋 Checklist لكل منتج قبل النشر

### **1. المقدمة (Introduction)**
- [ ] المقدمة عامة فقط (بدون موديلات محددة)
- [ ] لا تذكر Part Numbers في المقدمة
- [ ] لا تذكر الموديلات المتوافقة في المقدمة
- [ ] مثال صحيح: "كيبورد Dell Latitude هو الحل المثالي..."

### **2. أرقام القطع المتوافقة (Part Numbers)**
- [ ] Part Numbers موجودة في سيكشن "⚙️ أرقام القطع المتوافقة" فقط
- [ ] لا تكرر Part Numbers في المقدمة
- [ ] لا تكرر Part Numbers في المواصفات التقنية
- [ ] لا تكرر Part Numbers في blockquote

### **3. الأجهزة المتوافقة (Compatible Models)**
- [ ] الموديلات موجودة في سيكشن "💻 الأجهزة المتوافقة" فقط (في الجدول)
- [ ] لا تكرر الموديلات في المقدمة
- [ ] لا تكرر الموديلات في المواصفات التقنية (لا تكتب "التوافق: ...")
- [ ] لا تكرر الموديلات في blockquote

### **4. المواصفات التقنية (Technical Specs)**
- [ ] المواصفات التقنية فقط (Capacity, Voltage, Cells, Backlight, Language, Layout, Color)
- [ ] لا تكتب "التوافق: ..." في المواصفات التقنية
- [ ] لا تكتب "Part Number: ..." في المواصفات التقنية
- [ ] اللون: فقط لو مذكور في بيانات المنتج

### **5. الضمان (Warranty)**
- [ ] الضمان صحيح حسب نوع المنتج:
  - البطاريات: 3 شهور
  - الكيبوردات: شهر واحد
  - الشاشات: 3 شهور
  - الشواحن: 3 شهور
  - المراوح: شهر واحد
  - الهوسنج: لا تذكر الضمان نهائياً

### **6. خدمة التركيب (Installation Service)**
- [ ] موجودة في منتجات قطع الغيار فقط
- [ ] غير موجودة في المنتجات الرقمية
- [ ] HTML Template صحيح مع display: flex

### **7. Blockquote (تأكد من التوافق)**
- [ ] نص عام فقط (بدون موديلات محددة)
- [ ] نص عام فقط (بدون Part Numbers محددة)
- [ ] مثال صحيح: "تأكد من موديل الجهاز أو Part Number قبل الطلب..."

### **8. SEO**
- [ ] SEO Title: 50-60 حرف
- [ ] Meta Description: 150-160 حرف (بدون أسعار)
- [ ] Short Description: بدون أسعار
- [ ] Tags: 8-12 tags (كل الموديلات المتوافقة)

### **9. Attributes**
- [ ] Brand
- [ ] Model (بدون تكرار الموديلات كلها)
- [ ] Part Number (لو موجود)
- [ ] Backlight (للكيبوردات)
- [ ] Language
- [ ] Layout (للكيبوردات)
- [ ] Color (فقط لو مذكور)
- [ ] Condition: "Brand New - High Quality"
- [ ] Warranty (حسب نوع المنتج)

### **10. Images**
- [ ] على الأقل صورة واحدة
- [ ] Alt Text موجود لكل صورة
- [ ] الصور 1000x1000 بكسل

---

## 🔍 مراجعة سريعة:

1. **هل الموديلات المتوافقة مكررة في أكثر من مكان؟** ❌
2. **هل Part Numbers مكررة في أكثر من مكان؟** ❌
3. **هل المقدمة عامة بدون موديلات؟** ✅
4. **هل المواصفات التقنية بدون "التوافق"؟** ✅
5. **هل blockquote عام بدون موديلات/Part Numbers؟** ✅

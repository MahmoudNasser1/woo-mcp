# ⚡ Rank Math SEO - Quick Start Guide

## ✅ تم إضافة أدوات Rank Math SEO!

الآن يمكنك الوصول إلى Rank Math SEO والتعديل فيها مباشرة من خلال MCP.

## 🎯 الأدوات المتاحة

### 1. `get_rank_math_seo` - عرض بيانات Rank Math
### 2. `update_rank_math_seo` - تحديث بيانات Rank Math

---

## 📋 الاستخدام السريع

### عرض Rank Math SEO:
```
@woocommerce اعرض Rank Math SEO للمنتج 1015
```

### تحديث Rank Math SEO:
```
@woocommerce حدث Rank Math SEO للمنتج 1015:
- Title: "هاوسينج Lenovo Ideapad 110-15ISK - أفضل سعر | Fix Zone"
- Description: "هاوسينج كامل للابتوب Lenovo Ideapad 110-15ISK. جودة عالية، سعر مناسب."
- Focus Keyword: "lenovo ideapad 110 housing"
```

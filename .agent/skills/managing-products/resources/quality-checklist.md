# 📊 قائمة التحقق من الجودة (Expert Quality Check)

يجب الحصول على **90 نقطة** على الأقل للنشر.

| المعيار | الوزن | التعليمات |
|---------|-------|-----------|
| **📸 الصور** | **30** | - 3 صور على الأقل (خلفية بيضاء نقية).<br>- الأبعاد: 1000×1000px.<br>- بدون علامات مائية أخرى. |
| **📝 Alt Text** | **10** | - وصف دقيق لكل صورة (مثال: "بطارية Dell Latitude E5470 الأصلية"). |
| **🔍 SEO Title** | **15** | - الطول: 50-60 حرف.<br>- يتضمن: Brand + Model + Type + "Fix Zone". |
| **📄 Meta Desc** | **15** | - الطول: 150-160 حرف.<br>- يتضمن: Focus Keyword + Call to Action (اطلب الآن). |
| **🏷️ Tags** | **10** | - 8 tags على الأقل.<br>- تشمل: الموديل، البراند، رقم القطعة، النوع. |
| **⚙️ التوافق** | **20** | - جدول Models كامل ودقيق.<br>- جدول Part Numbers كامل.<br>- عدم ذكر موديلات غير متوافقة. |
| **المجموع** | **100** | |

### � أخطاء تمنع النشر فوراً (Critical Blockers):
1.  **السعر صفر أو خطأ:** تأكد من العملة (EGP).
2.  **عنوان غير مفهوم:** عناوين مثل "Battery Laptop" مرفوضة.
3.  **صور غير واضحة:** صور مبكسلة أو مظلمة.
